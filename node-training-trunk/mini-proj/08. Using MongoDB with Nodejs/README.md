##### Get mongodb
```
mongodb.com >> download-center >> Server
``` 

##### Default Port	
```
27017 or 27018
``` 

##### Run it (if it is not running automatic)
```
Program Files\MongoDB\Server\4.0\bin\mongod.exe
``` 

##### You can use as a GUI (self tested)
```
MongoDB Compass or Studio 3T
```
